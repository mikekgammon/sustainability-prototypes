ℹ Design approach: Otto said to design this as the same system as Scope 3, not a separate feature. The underlying data model is identical — activity type, unit, carbon factor, cost per unit. The difference is that Alternative Fuel projects replace an existing fuel source (fuel switching), whereas Scope 3 adds a new category. These share the same wizard and the same activity step. Two screens differ: the fuel type selector and the fuel switch comparison view. Requires Otto's Asana ticket review before dev build.
