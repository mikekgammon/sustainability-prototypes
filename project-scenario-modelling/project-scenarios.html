Wireframe — Low fidelity · Scenario Modelling v0.1

This wireframe replaces the current Scenario Planning panel with an interactive energy cost curve editor. Users drag existing data points to reshape the forecast over the project lifespan, and zoom into any time window for monthly detail. The chart updates in real time and the impact summary reflects the change.
